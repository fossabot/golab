## golab user ssh-keys get

Single SSH key

### Synopsis


Get a single SSH key for a given user.

```
golab user ssh-keys get [flags]
```

### Options

```
  -h, --help     help for get
  -i, --id int   (required) key id of SSH key to be shown
```
